## Descripcion
Primera prueba practica de la materia Tecnicas de Programacion. Desarrollo de aplicacion web con React que demuestra los conceptos basicos aprendidos.

## Objetivos
- Crear componentes funcionales en React
- Manejar estado con Hooks
- Implementar ciclo de vida de componentes
- Estilizar con CSS

## Tecnologias Utilizadas
- React 18.2.0
- JavaScript ES6+
- CSS3
- Git/GitHub

## Evaluacion
Esta prueba evalua:
- Correcta estructura de componentes
- Manejo de props y state
- Sintaxis JavaScript/JSX correcta
- Organizacion del codigo
